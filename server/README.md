# Developer Setup

## Install Poetry
* `curl -sSL https://install.python-poetry.org | python3 -`
## One-Time
* `poetry install`
## Add new dependency 
* `poetry add ...`
## Useful commands
* `poetry show --tree`
* `poetry show --outdated`

## Run
* `poetry shell`
* `poetry run python browser_automation.py`

## Unit tests
* pytest
